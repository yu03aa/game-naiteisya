# Turn-Based RPG Game

ブラウザベースのターン制RPGゲーム

## セットアップ

### 必要環境

- Node.js 16+
- npm または yarn

### インストール

```bash
npm install
```

### 開発サーバー起動

```bash
npm run dev
```

### ビルド

```bash
npm run build
```

### プレビュー

```bash
npm run preview
```

## 遊び方

1. 職業を選択（戦士、魔法使い、盗賊、僧侶）
2. 「冒険に出発」でバトル開始
3. 攻撃、防御、スキル、アイテムを駆使して敵を倒す
4. レベルアップしてさらに強敵に挑戦！

## 技術スタック

- React 18
- Vite
- Tailwind CSS
- Lucide React (Icons)
